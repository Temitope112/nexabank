# NexaBank Dashboard

A **frontend fintech dashboard prototype** simulating a digital banking interface built with **HTML, CSS, and JavaScript**.  
Designed as a portfolio project to showcase frontend skills, responsive design, and interactive UI components.

## Live Demo
[View on Netlify]https://steady-banoffee-83e8a0.netlify.app/

## Features
- Dynamic account balance display
- Transaction feed with credit/debit distinction
- Transfer modal simulation
- Responsive layout for desktop and mobile
- Interactive chart showing balance trend over time

## Technologies Used
- HTML5
- CSS3
- JavaScript
